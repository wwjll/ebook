
在windows上配置nodejs和nvm：
https://www.cnblogs.com/wyy1234/p/9727142.html
步骤：
1.修改nvm下setting.txt，设置path为nodejs的安装位置
2.修改nvm环境变量：
NVM_HOME:E:\dev\nvm
NVM_SYMLINK:E:\dev\nodejs
PATH:%NVM_HOME%;%NVM_SYMLINK%（path已存在【%NVM_HOME%;%NVM_SYMLINK%】在添加到最后）
3.安装全局npm
npm config set prefix "E:\dev\nvm\npm"//配置用npm下载包时全局安装的包路径
npm install npm -g --registry=https://registry.npm.taobao.org
在用户变量中修改 NPM_HOME=E:\dev\nvm\npm

一些替代npm的方式
npm install -g cnpm --registry=https://registry.npm.taobao.org //使用淘宝镜像cnmp替代npm
npm install -g yarn//使用yarn替代npm

npm 常见命令
安装指定版本：在其后加 ‘@版本号’ 例：npm install xxx@1.1.1

查看具体模块：npm list 0
